The form below is a Thymeleaf fragment. The submit is done in Javascript (AJAX), the HTML5 validation is fired manually and the result is directly displayed without reloading the page : only the form fragment is reloaded. To have a such process, we initialize the form model with ajax submit option in this page controller.

Note : the Post-Redirect-Get pattern is still viable : the browser first do a POST, get the answer and if the form is valid, load the page (in JS) defined with the successRedirect form field.
