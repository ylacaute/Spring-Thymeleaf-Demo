Post-Forward

This form has been created with a forward attribute : when the form is submited, Spring forward to this page controller. This is faster, but you will have to rewrite your URL.

Post-Redirect-Get

This form has been created with a redirect attribute : when the form is submited, Spring answer a redirect to the browser who will make a new request to this page. The URL will be as expected. This is the common way to do. The good stuff here is that we have a specific form controller, which is only linked to the form fragment, not the page which use this fragment.
